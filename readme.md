# Sistem Informasi Perpustakaan (PHP Native)

Sistem Informasi Perpustakaan merupakan aplikasi berbasis web yang dibangun menggunakan **PHP Native** untuk membantu pengelolaan data perpustakaan secara terkomputerisasi.  
Aplikasi ini mendukung pengelolaan data mahasiswa, petugas, buku, peminjaman, pengembalian, serta perhitungan denda keterlambatan.

Project ini dibuat untuk keperluan **tugas/UAS** dan sebagai media pembelajaran pengembangan aplikasi web berbasis PHP tanpa framework.

---

## Fitur Utama
- Autentikasi pengguna (Login dan Logout)
- Manajemen data mahasiswa
- Manajemen data petugas
- Manajemen data buku
- Peminjaman buku
- Pengembalian buku
- Perhitungan denda keterlambatan
- Pencetakan laporan dalam format PDF

---

## Teknologi yang Digunakan
- PHP (Native)
- MySQL / MariaDB
- HTML dan CSS
- Bootstrap
- DOMPDF (untuk pembuatan laporan PDF)
